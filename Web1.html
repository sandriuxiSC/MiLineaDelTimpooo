Â¿Quién la creo?

En 1991 se publica la primera página web creada por Tim Berners-Lee usando un computador NeXT. Esta página,hablaba sobre el emergente y emocionante mundo del World Wide Web. Es la forma más básica que existe, con navegadores de sólo texto bastante rápidos. Después surgió el HTML que hizo las páginas web más agradables a la vista, así como los primeros navegadores visuales tales como IE, Netscape, Explorer (en versiones antiguas), etc. La Web 1.0 es de sólo lectura. El usuario no puede interactuar con el contenido de la página (no puede comentar ni responder directamente), estando totalmente limitado a lo que el Webmaster sube a ésta.
